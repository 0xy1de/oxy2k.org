LinX OS

LinX OS is a retro‑modern Linux distribution that pairs a classic early‑2000s look with modern security and drivers. The default theme is LinX Aero running on Oxyde.

  • Fast boot, light footprint
  • Aqua‑inspired window chrome and taskbar
  • Modern kernel, old‑school charm
Goal: friendly on older hardware yet crisp on new.
Oxyde Desktop

Oxyde is a Rust‑powered desktop environment and compositor aiming for under 1 GB total RAM with buttery animations and tasteful blur/glass.

  • Rust core • safety + speed
  • Hand‑tuned effects (Aqua glass, gentle shadows)
  • Classic key: Win opens the app drawer
License: MPL‑2.0 (WIP modules may vary).
